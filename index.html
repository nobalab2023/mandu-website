About MANDU

MANDU is a BEP-20 utility token in the NOVA LAB ecosystem. The project focuses on payment and rewards use-cases with planned integration to UB Pay (biometric payment) and Mandu Messenger. This site provides neutral, verifiable information for users and reviewers.

Token Overview

  • Name: MANDU

  • Standard: BEP-20 (BNB Smart Chain)
  • Contract: 0x685a2ff6Cb8aE62Fbc56d932A7C96ef605768036
  • Total Supply: 80,000,000

Utility

  • Payments and transfers (via UB Pay; planned)
  • In-app purchases and rewards (Mandu Messenger; planned)
  • Merchant programs & community incentives
No investment return is promised. This token is intended for utility within the ecosystem.
